Our Mission

A Modern Library for a Modern Campus

The Premier University Library Management System was built as an Internet Programming Lab project to demonstrate full-stack web development skills using HTML, CSS, JavaScript, and local data persistence.

Our goal was to create a real, usable system that benefits students, teachers, and administrators — replacing manual paperwork with a seamless digital experience.

  • Built for Premier University's academic community
  • Multi-role access: Admin, Teacher, Student
  • Real CRUD operations for books and materials
  • XML-powered notice board with JSON conversion
  • Responsive design for all screen sizes
  • LocalStorage + SessionStorage data persistence
